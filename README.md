# Premium Portfolio

Personal portfolio site built with React, Vite, Tailwind CSS, and Framer Motion.

## Requirements

- Node.js 20+
- npm 10+

## Run Locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite (usually `http://localhost:5173`).

## Build For Production

```bash
npm run build
npm run preview
```

## Quality Checks

```bash
npm run lint
npm run test
```

## Contact Links

Update contact details in [src/App.jsx](./src/App.jsx):

- `contact.email`
- `contact.github`
- `contact.linkedin`

The contact form uses `mailto:` and opens the default email client.

## Deploy

Any static host works (Vercel, Netlify, Cloudflare Pages, GitHub Pages).

- Build command: `npm run build`
- Publish directory: `dist`
